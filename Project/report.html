Bachelor of Science in Applied Health Sciences

https://applebaum.wayne.edu/bsahs

I chose the Bachelor of Science program offered by the Applebaum college in Applied Health Sciences.
This site fits the requirements of the assignment. It has a well defined header and footer; over 25
links; a main content area; 3 distinct lists; and one image. And unlike many of the other program
sites I perused, it has at least 3 heading levels.

Major Sections

Site header

<header>

As this program page is one of many at Wayne State University, it is partcularly appropriate
to use the HTML5 semantic element of <header>. Not only does this provide uniform
branding by employing the Wayne State University colors and logo, but it is an efficient way to anchor
links to help the user traverse the entirety of the Wayne State University site.

Main Navigation

<nav>

As this section is used to traverse additional pages relating to the specific program of Applied Health
Sciences, the <nav> HTML5 semantic element is most appropriate. Using this element is
crucial for search engine optimization and accessibility due to recognizability for screen readers.

Primary Content

<main>

Using the <main> HTML5 semantic element is an effective way to provide structure to the
HTML code for both readability as well as the purposes of editing. In addition, using the structural element
enhances the effectiveness of indexing and accessiblity.

Page Footer

<footer>

Again, due to this program being one of the many at Wayne State University, the branding inherent in the
semantic element of the <footer> is useful for evoking a feeling of cohesion in the user.
This is also a good element with which to anchor program specfic contact information as well as external social
media links.

Headings

The site uses 4 levels of headings, located within the

<main>
<footer>

elements, though I would only use 3.

The following are the text and the corresponding levels as moving through in document order:

  1. "Bachelor of Science in Applied Health Sciences" - H1.
    This should be the first level as it is the subject of the entire page.

  2. "Building a Strong Foundation for a Successful Future" - H2.
    This should be the next level down, as it is a subset of the program in level H1.
    (It should be noted that the website codes this as level H3 in the HTML)

  3. "Graduates will emerge with a strong foundation in:" - H3.
    This is a direct subset of the previous section.
    (It should be noted that the website codes this as level H4 in the HTML)

  4. "Direct Admission to WSU Applebaum" - H2.
    This is not a subset of the previous H2, so it should become another H2, as a subset of the H1 program.
    (It should be noted that the website codes this as level H3)

  5. "Attend an Information Meeting" - H2.
    As this a subset of the program, it should be another H2.
    (It should be noted that the website codes this as level H3)

  6. "Additional Information" - H2.
    As this is a subset of the program, it should be another H2.
    (It should be noted that the website codes this as level H3)

  7. "Bachelor of Science in Applied Health Sciences" - H2
    Though this is the name of the program, as it is in the footer element, it should be less prominent.

  8. "Eugene Applebaum College of Pharmacy and Health Sciences" - H3
    As this is not the specific program, it should be down one level from the program name next to it in the footer.

  9. "Resources" - H3
    This should have the same prominence as what is next to it.

Links

Internal Links

There are several internal links listed in the Main Navigation section, including "About" and "FAQ".
I would expect these links to open in the same tab, as the pages are located within Wayne State University.

External Links

There are several external links located in the page footer section, including "Facebook" and "Instagram".
I would expect these links to open in a new tab, since the user is being taken outside of the University,
and Wayne State should ensure that users stay connected to the University site. These links are indicated with
visual icons.

Accessibility Review

Upon analyzing the accessiblity of the site by using the WAVE there were 6 problematic findings.

  • 1 Error
  • 5 Alerts

Of the findings the two most problematic were:

  1. Broken skip link
  2. Skipped heading level

The page does have Null or empty alternative text for the embedded image.
One change that could improve accessibilty, would be to reorder the heading levels.

HTML Validation

Upon validating the HTML code of the site by using the W3C HTML Validator there were 5 problematic findings.

  • 4 Errors
  • 1 Warning

The error found in the 16th listed entry, indicates that at line 216 of the code an H3 level heading appears
directly after an H1 level heading, thus skipping 1 level. One way to solve this would be to reorder the headings.

Summary

Overall, the page for the "Bachelor of Science in Applied Health Sciences" program is well designed. It is well
structured in its utilization of proper HTML5 semantic elements. The page's effective design yields a mostly successfully
validated and accessible website. Fixing the single accessiblilty error of the broken skip link would eliminate all
errors of this kind. In addition, reording the heading levels would alleviate confusion in the structure.
